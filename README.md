# 📊 Function Treat

## 📌 Project Description
The **Data Analyzer and Transformer Program** is a Python console application that allows users to input numerical data and perform different operations such as data analysis, factorial calculation, filtering, sorting, and statistical analysis.

This project demonstrates the use of:
- Lists
- Loops
- Functions
- Recursion
- Lambda Functions
- Filter()
- Sorting
- Match-Case Statement
- User Input

---

# ✨ Features

## 1. Input Data
- Accepts integer values from the user.
- Stores the values in a 1D list.

Example:
```
Enter data:
10 20 30 40 50
```

---

## 2. Display Data Summary
Displays:

- Total Elements
- Minimum Value
- Maximum Value
- Sum of Elements

Example:
```
Total Elements : 5
Minimum Value : 10
Maximum Value : 50
Sum : 150
```

---

## 3. Calculate Factorial
Calculates the factorial of a number using **Recursion**.

Example:
```
Input : 5

Output :
Factorial = 120
```

---

## 4. Filter Data by Threshold
Shows only those values that are greater than or equal to the entered threshold.

Example:
```
Data :
10 15 22 35 40

Threshold :
20

Output :
22 35 40
```

---

## 5. Sort Data

Two sorting options are available:

### Ascending Order
```
10 20 30 40 50
```

### Descending Order
```
50 40 30 20 10
```

---

## 6. Display Dataset Statistics

Displays

- Minimum
- Maximum
- Sum
- Average

Example

```
Minimum : 10
Maximum : 50
Sum : 150
Average : 30.0
```

---

## 7. Exit Program

Ends the application.
